## ① Qu'est-ce qu'Internet ?

> **Internet ≠ le Web !**
> Le Web n'est qu'un moyen de l'utiliser (parmi d'autres).

Internet = **PROTOCOLE** = un ensemble de règles

### 3 niveaux

1. **Base matérielle / décentralisée** (serveurs / routeurs)
   - ~45 000 réseaux autonomes interconnectés

2. **Ensemble de protocoles** (OPEN SOURCE)
   - ~6000 normes et protocoles
   - Pile TCP/IP → internet / différents niveaux physiques
   - Régulation : UCI

3. **Un univers d'applications**
   - Des milliards d'utilisateurs

---

## ② Pourquoi Internet est décentralisé ?

- **Télégraphe** = proto-Internet → paquets / semi-numérique / pseudo bits → **Routage décentralisé**
- **Téléphone** = Commutation par circuits / signal analogique → **Routage centralisé**

### Comment est prise la décision ?
- Pas d'autorité centrale
- Chaque "carrefour" / "nœud" a la capacité d'agir de manière autonome
