# Qu'est-ce que le Web ?
**Semaine 1 · Jour 2 · Page 5**
Tags: #alyra #web1 #web2 #web3 #blockchain #decentralisation

---

## ③ Le Web = ?

Le Web est à la fois :
- → Un **service**
- → Un **format de données**
- → Un **langage** (interagir)
- → Un **protocole**
- → Un **tournant des applications**

---

## ★ Web 1

Architecture classique en 3 couches :
- → **Données** (BDD)
- → **Logique de contrôle** (Back-end)
- → **Ce que l'on voit** (Front-end)

---

## ★ Web 2

- → **Forums** (login / compte)
- → **Réseaux sociaux**

Interaction et participation des utilisateurs.

---

## ★ Web 3

> Possession de la donnée — **Publique** / "non privée" (Blockchain)

- → **Écriture** de la donnée publique → *Smart Contracts, Consensus*
- → **Lecture** de la donnée publique → *Interface Web*
