## Principe Client — Serveur

- La **première** machine qui parle = le **client**
- La **deuxième** machine = le **serveur**

---

## P2P = Peer to Peer ♥

> Les machines sont à la fois **client ET serveur**

Réseau maillé — chaque nœud peut communiquer avec tous les autres directement.

---

## ④ DNS — Domain Name System (1984)

Le DNS traduit une adresse lisible en adresse IP.

**Exemple Google :**
- IP réelle : `173.194.78.94`
- Accès brut : `http://173.194.78.94` → (annuaire centralisé)
- Accès humain : `www.google.fr` ← **DNS**

### Structure d'une URL

```
http:// www . free . fr
  ↑      ↑     ↑     ↑
protocole label 2nd   Top Level
               level  Domain
               domain
```
← sens de lecture (droite → gauche)
