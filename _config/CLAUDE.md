# CLAUDE.md — Vault Obsidian Xavier

## Rôle de ce vault
Vault de travail pour la formation Alyra DeFi (Promo Blaise Pascal) et la recherche personnelle.
Sert à la fois de base de connaissances, de journal de stratégies et d'espace de réflexion.

## Structure
- `Raw/` : notes brutes, clippings, ressources de formation — pas encore traitées
- `Wiki/` : fiches de référence stables sur les mécanismes, protocoles, actifs, concepts
- `Strategies/` : une note par stratégie DeFi (active, en cours d'analyse, archivée)
- `Data/` : données de marché et analyses par catégorie de protocole
- `_config/` : configuration du vault (ce fichier, style, templates)

## Conventions
- Langue des notes : français (sauf termes techniques DeFi qui restent en anglais)
- Liens internes Obsidian avec `[[double crochets]]`
- Tags avec `#tag` en bas de note
- Frontmatter YAML pour les notes Wiki et Strategies

## Ce que Claude doit savoir
- Ne pas modifier les notes existantes sans demande explicite
- Toujours proposer le contenu avant d'écrire une nouvelle note
- Pour les fiches Wiki : utiliser le template `_config/templates/`
- Les notes Raw sont des brouillons — ne pas les reformater automatiquement
