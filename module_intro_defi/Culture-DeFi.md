## CT (Crypto Twitter / Crypto X)

Le réseau social **principal** de l'écosystème DeFi. Tout s'y passe : alpha, débats, hacks, mèmes.

**Comment utiliser CT :**
- Suivre les comptes des protocoles (@AaveAave, @Uniswap...)
- Suivre les chercheurs en sécurité (@tayvano_, @0xfoobar)
- Suivre les dégens (@0xGeeGee, @DegenSpartan...)
- Créer des listes thématiques
- Activer les notifications pour les comptes importants

---

## Comptes Anonymes

La **pseudonymité est la norme** en DeFi. Des protocoles majeurs ont été créés par des équipes entièrement anonymes (Curve — "Michael Egorov" connu seulement plus tard).

Exemples de pseudonymes influents :
- @0xGeeGee (analyste DeFi)
- @DegenSpartan (ancien fund manager anonyme)
- @Tetranode (whale Ethereum)

## Vocabulaire DeFi essentiel

| Terme | Signification |
|---|---|
| **DYOR** | Do Your Own Research — faites vos propres recherches |
| **WAGMI** | We're All Gonna Make It — optimisme collectif |
| **NGMI** | Not Gonna Make It — quelqu'un qui fait une erreur |
| **Rekt** | Ruiné, liquidé |
| **Ape in** | Investir sans vraiment analyser |
| **Degen** | Degenerate — investisseur à risque élevé (auto-dérision) |
| **GM / GN** | Good Morning / Good Night — salutations rituelles |
| **Ser** | Sir — forme respectueuse humoristique |
| **Fren** | Friend — camarade de la communauté |
| **Alpha** | Information exclusive/avant-gardiste |
| **Rugged / Rugpull** | Projet arnaque qui disparaît avec les fonds |
| **Moon** | Prix qui monte très fort |
| **Pump & Dump** | Manipulation de prix |
| **Whale** | Gros détenteur de crypto |
| **Diamond hands** | Tenir ses positions sans vendre |
| **Paper hands** | Vendre à la moindre baisse |
| **Ser this is a Wendy's** | Réponse humoristique face à une question trop technique |

## Design DeFi : deux écoles

**École minimaliste / hacker :**
- Curve Finance — texte vert sur fond noir, style terminal Unix
- Uniswap — épuré, minimaliste

**École colorée / anime / meme :**
- PancakeSwap — lapin astronaute, couleurs vives, GameFi
- Thena — guerrière cyberpunk néon sur BNB Chain
- YAM Finance — émojis, "fair finance by the people"

## Meme culture

Les mèmes sont un **vecteur de communication sérieux** dans l'écosystème. Ils:
- Expliquent des concepts complexes en images
- Commentent les hacks (souvent avec humour noir)
- Créent de la cohésion communautaire

Exemple classique : Mèmes Simpson, Drake, "This is fine" dog...
