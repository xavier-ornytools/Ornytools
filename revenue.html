Comment vérifier et saisir
💰 GMX — Automatique SYNC ↑
Utilise le bouton ⚡ Sync ci-dessus — les données arrivent directement depuis la blockchain.
→ app.gmx.io/#/referrals (onglet Affiliates pour vérifier)
🔄 1inch — Manuel WALLET
Les revenus 1inch arrivent directement en USDC ou ETH sur ton wallet Rabby.
1. Va sur → DeBank — ton wallet
2. Onglet History → cherche les transactions entrantes depuis 1inch
3. Note le montant → sélectionne 1inch dans le formulaire → entre le montant
⚡ Paraswap — Manuel WALLET
Les revenus Paraswap arrivent en ETH ou token natif sur ton wallet.
1. Va sur → Arbiscan — ton adresse
2. Onglet ERC-20 Token Txns → cherche les tokens reçus depuis Paraswap
3. Note le montant → sélectionne Paraswap → entre le montant
☕ Ko-fi — Manuel EMAIL
Tu reçois un email automatique à chaque don + notification dans Ko-fi.
1. Va sur → ko-fi.com/manage/donations
2. Copie le montant reçu (après commission Stripe ~2.9%)
3. Sélectionne Ko-fi dans le formulaire → entre le montant net
