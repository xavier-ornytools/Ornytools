## Coin vs Token : quelle différence ?

### Coin
> Un **coin** est la cryptomonnaie **native** d'une blockchain L1.

- Émise directement par la blockchain
- Sert à **payer les frais de transaction** (gas)
- Créée pour **récompenser les validateurs** (mineurs POW ou stakers POS)

**Exemples :**
- **BTC** = coin de la blockchain Bitcoin
- **ETH** = coin de la blockchain Ethereum

---

### Token
> Un **token** est une cryptomonnaie **émise sur** une blockchain existante (via un smart contract).

- N'importe qui peut en créer un
- Usages quasi illimités

**Exemples :**
- **USDC** (USD Coin) → stablecoin émis par Circle
- **AAVE** → token de gouvernance du protocole Aave

---

## Usages d'un token

| Type d'usage | Exemple |
|-------------|---------|
| Représenter un actif | Stablecoin (USDC, DAI) |
| Tokénisation | Immobilier, actions tokenisées |
| Propriété d'une œuvre | NFT (art, musique...) |
| Gouvernance | Voter sur un protocole (AAVE, UNI...) |
| Accès à un protocole | Token utilitaire |

---

## Fongibilité

| Type | Description | Exemple |
|------|-------------|---------|
| **Fongible** | Divisible, indiscernable (1 ETH = 1 ETH) | ETH, USDC, BTC |
| **Non-fongible (NFT)** | Indivisible, numéroté, unique | CryptoPunks, BAYC |

---

## Résumé : Coin vs Token

| | **Coin** | **Token** |
|-|---------|---------|
| Émis par | La blockchain L1 elle-même | Un smart contract sur une blockchain |
| Qui peut en créer ? | Seule la blockchain | N'importe qui |
| Usage principal | Frais de transaction + récompenses | Infini (gouvernance, actifs, NFT...) |
