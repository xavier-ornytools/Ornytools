# 🔤 DNS
Tags: #concept #réseau #internet

---

## Définition
**DNS** (Domain Name System) est l'annuaire d'Internet : il traduit les noms lisibles (`google.com`) en adresses IP numériques (`142.250.74.46`).

---

## Fonctionnement simplifié
1. Tu tapes `uniswap.org` dans ton navigateur
2. Ton ordinateur interroge un serveur DNS
3. Le DNS répond avec l'adresse IP correspondante
4. La connexion s'établit

---

## Problème central
> Le DNS est **centralisé** — contrôlé par des registraires et gouvernements

- Peut être censuré (gouvernements bloquent des domaines)
- Peut être falsifié (DNS poisoning)
- Point de défaillance unique

---

## Alternative Web3
- **ENS** (Ethereum Name Service) → `.eth` comme `vitalik.eth`
- Noms de domaine stockés sur la blockchain, non censurables
- Résout le même problème que DNS mais de façon décentralisée
