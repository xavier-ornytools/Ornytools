# 🏦 DeFi — Finance Décentralisée
Tags: #concept #defi #ethereum

---

## Définition
La **DeFi** (Decentralized Finance) est un ensemble de services financiers construits sur la [[Blockchain]], accessibles à tous, sans intermédiaire (banque, courtier, assurance).

> La DeFi reproduit les services bancaires traditionnels avec des [[Smart-Contracts]].

---

## DeFi vs Finance Traditionnelle (TradFi)

| | TradFi | DeFi |
|--|--------|------|
| **Intermédiaire** | Banque, courtier | Smart contract |
| **Accès** | KYC obligatoire | Wallet suffit |
| **Disponibilité** | 9h-17h semaine | 24h/24 365j/an |
| **Transparence** | Opaque | Code open source |
| **Garde des fonds** | Banque | Toi-même |
| **Censure** | Possible | Difficile |

---

## Primitives DeFi

### 1. DEX — Exchange décentralisé
- Échange de tokens sans intermédiaire
- Exemples : **Uniswap**, Curve, Balancer
- Mécanisme : AMM ([[Liquidité]] et pools)

### 2. Lending / Borrowing
- Prêt et emprunt de crypto sans banque
- Exemples : **[[Aave]]**, Compound, MorphoBlue
- Collatéral obligatoire → liquidation si Health Rate trop bas

### 3. Yield Farming
- Déposer des actifs pour générer des rendements
- Risque : smart contract, impermanent loss

### 4. Stablecoins
- **DAI** (MakerDAO) → décentralisé, collatéralisé en crypto
- **USDC / USDT** → centralisés mais très utilisés

### 5. Derivatives
- Contrats à terme, options décentralisées
- Exemples : GMX, dYdX

---

## Risques DeFi

| Risque | Description |
|--------|-------------|
| **Smart contract** | Bug dans le code → hack |
| **Liquidation** | Collateral insuffisant → position liquidée |
| **Impermanent loss** | Perte en LP lors de variation de prix |
| **Oracle manipulation** | Prix falsifié → exploit |
| **Rug pull** | Équipe qui part avec les fonds |
