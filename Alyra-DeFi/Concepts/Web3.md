# 🌐 Web3
Tags: #concept #web #décentralisation

---

## Définition
**Web3** désigne la prochaine évolution d'Internet, basée sur la [[Blockchain]] et la décentralisation, où les utilisateurs contrôlent leurs données et actifs.

---

## Les 3 ères du Web

| | Web 1.0 | Web 2.0 | Web 3.0 |
|--|---------|---------|---------|
| **Période** | 1991–2004 | 2004–2020 | 2020→ |
| **Lecture** | ✅ | ✅ | ✅ |
| **Écriture** | ❌ | ✅ | ✅ |
| **Propriété** | ❌ | ❌ | ✅ |
| **Données** | Statiques | Plateformes | Blockchain |
| **Exemples** | Sites statiques | Facebook, Google | Uniswap, ENS |

---

## Principes du Web3

- **Décentralisé** → pas de serveur central ([[P2P]])
- **Sans permission** → tout le monde peut participer
- **Trustless** → confiance dans le code, pas dans les humains
- **Propriété native** → NFT, tokens = vrais actifs numériques

---

## Composants clés

- [[Blockchain]] → registre immuable partagé
- [[Smart-Contracts]] → logique applicative autonome
- **Wallets** → identité et propriété (MetaMask, Rabby)
- **IPFS** → stockage décentralisé des fichiers
- [[DNS]] / **ENS** → noms de domaine décentralisés

---

## Critique
> Web3 n'est pas encore totalement décentralisé en pratique

- Beaucoup de dApps utilisent encore des API centralisées (Infura, Alchemy)
- UX encore difficile pour le grand public
- Scalabilité en cours de résolution (L2)
