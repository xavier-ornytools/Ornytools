# 🌐 TCP/IP
Tags: #concept #réseau #internet

---

## Définition
**TCP/IP** (Transmission Control Protocol / Internet Protocol) est la suite de protocoles fondamentale qui permet la communication sur Internet.

- **IP** → adresse et achemine les paquets de données d'une machine à l'autre
- **TCP** → garantit que les paquets arrivent dans l'ordre et sans perte

---

## Modèle en couches

| Couche | Rôle | Exemple |
|--------|------|---------|
| Application | Ce que l'utilisateur voit | HTTP, SMTP |
| Transport | Découpe et réassemble les données | TCP, UDP |
| Internet | Routage entre réseaux | IP |
| Réseau | Transmission physique | Ethernet, Wi-Fi |

---

## Points clés
> TCP/IP est **décentralisé** par design — aucun point de contrôle unique

- Créé dans les années 1970 par DARPA
- Base sur laquelle tout Internet (et donc la blockchain) est construit
- La blockchain hérite de cette logique de décentralisation
